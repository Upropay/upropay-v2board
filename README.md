# Upropay V2Board Payment Plugin

USDT（TRC20）支付插件，适用于 V2Board。
Plug and play. No coding required.

## 🚀 Features / 功能特点

* ✅ Support USDT (TRC20) / 支持 USDT TRC20
* ✅ Auto order callback / 自动回调订单
* ✅ Easy integration / 即插即用
* ✅ Secure API signature / 安全签名验证
* ✅ Lightweight & open source / 轻量开源
* ✅ No modification required / 无需修改 V2Board 核心

## 📦 Installation / 安装教程

1. Download this repository / 下载本仓库
2. Upload `Upropay.php` to / 上传文件到：

/app/Payments/

3. Go to V2Board Admin Panel / 进入后台
4. Enable **Upropay** payment method / 启用插件
5. Fill in API credentials / 填写 API 信息

Done ✅ / 完成

## 💳 How It Works / 工作流程

1. User creates order / 用户创建订单
2. Plugin requests Upropay API / 插件请求接口
3. Payment address generated / 生成支付地址
4. User pays USDT / 用户支付
5. Upropay sends callback / 回调通知
6. Order marked as paid / 订单完成
